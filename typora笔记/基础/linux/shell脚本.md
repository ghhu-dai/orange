 

  

# 学习一门新语言

## shell语法：

数据类型：字符串string（shell只有一种数据类型，默认变量都是字符串）

### 变量：

* 全局变量 ---- 环境变量
* 局部变量 --- 本地变量
* 取变量的值 ：`${变量名}`，
  * 特别的取命令执行结果 ：`$(命令)`,
  * 对变量执行算术运算：`$((变量))`

#### 

转义字符 | 换行符：`\`

 单引号：括字符串，不能展开变量

双引号：括字符串，可以展开变量

















### 控制语句：

* if ,else
* switch case
* for,while

是否支持面向对象：继承、重载、封装、多态。。。
